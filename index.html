ROP::Ropeman

  • In this lab, students can get first-hand experience on exploiting stack buffer-overflow vulnerability with different ROP techniques.
  • In a series of stepwise challenges, students will meet different scenarios after finding out a stack overflow vulnerability, each requires different methods and skills to be exploited.
  • This repo is a part of the project SRCTF
  • To use this challenge sequence, please make sure you have set up the server framework, then follow the instructions to import the challenge contents.
